Account recovery

Forgot your password?

How to reset your password

  1. Contact an Axiom admin directly (via phone, chat, or other channel)
  2. Tell them your account email and ask for a password reset
  3. Admin will reset it from the Admin panel and give you a temporary password
  4. Log in with the temporary password
  5. Go to Profile → Change password and set a new one
